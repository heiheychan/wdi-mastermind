Master Mind

Instructions

  • Two players to start the game and take turn
  • Four color balls will be randomly generated as the answer
  • Each turn, player get to pick four color dots which hopefully look the same as the answer
  • At the end of each turn, hints will be provided:
    • green colored dot for correct color with correct place
    • yellow colored dot for correct color but wrong place
    • red colored dot for utterly wrong answer!
  • Have fun!
